# Installation

Maestro can be installed and run in several ways depending on your use case. Pick the one that fits your workflow.

| Method | Best for | Requirements |
|---|---|---|
| [Homebrew](homebrew) | End users, quick setup | macOS/Linux, Docker |
| [Docker](docker) | Production, CI/CD, containers | Docker |
| [Local Development](local-dev) | Contributing, debugging, extending | Python 3.12+, Node.js 22+, uv, Docker |

All methods need Docker running for PostgreSQL (or an external PostgreSQL instance).
